# Guia do Usuário Final

## O que é endereço público

Endereço público é como o número da sua conta bancária: pode ser compartilhado para receber
créditos, não dá acesso direto à carteira e não precisa de segredo.

## O que é frase de recuperação

A frase de recuperação (12 ou 24 palavras) permite recuperar toda a carteira.
Ela é sigilosa e não pode ser enviada para suporte.

## O que nunca compartilhar

- Frase de recuperação.
- Seed.
- Chave privada.
- Arquivos de backup não confiáveis.

## Diferença entre testnet e mainnet

- `testnet` é rede de testes. Os fundos não têm valor real.
- `mainnet` é rede real e pode envolver dinheiro de verdade.

## O que fazer se alguém viu a seed/frase

Considere a carteira comprometida.
Revogue imediatamente, migre para nova frase e, para valores altos, use hardware wallet.
