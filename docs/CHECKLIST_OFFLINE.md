# Checklist Offline

Imprima ou leia antes de gerar endereco.

- [ ] Wi-Fi desligado.
- [ ] Cabo de rede desconectado, se houver.
- [ ] Sem compartilhamento de tela.
- [ ] Sem camera apontada para o monitor.
- [ ] Sem gravacao.
- [ ] Sem print.
- [ ] Sem backup em nuvem.
- [ ] Dono da carteira presente.
- [ ] Suporte nao consegue ver a frase de recuperacao.
- [ ] Frase anotada em papel pelo dono.
- [ ] Endereco publico conferido.
- [ ] Rede conferida: Ethereum/EVM, Bitcoin testnet `tb1` ou Bitcoin mainnet `bc1`.
- [ ] Para valores altos, hardware wallet ou multisig foi recomendado.
